# Lost In Maze

Lost In Maze is a Unity 3D maze game where the player navigates through obstacles and challenges to reach the finish line.

## Features

* Player movement controls
* Cinemachine camera follow system
* Obstacles and hazards
* Finish line detection
* Custom materials and prefabs

## Built With

* Unity
* C#

## How to Play

1. Start the game.
2. Navigate through the maze.
3. Avoid obstacles.
4. Reach the finish line to win.

## Project Status

This project was developed as a learning and academic project using Unity.
